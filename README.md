# Solicitar

**Solicitar** is a modern job application tracking application that helps job seekers organize, manage, and monitor their job search from a single platform.

The application enables users to:

* Track job applications through customizable hiring stages.
* Manage interview schedules and important deadlines.
* Store resumes, cover letters, and supporting documents.
* Record notes and follow-up activities for each opportunity.
* Visualize job search progress with an intuitive dashboard.

Built with a clean, user-friendly interface, Solicitar is designed to streamline the job search process and improve productivity across mobile and web platforms.

### Tech Stack

* FlutterFlow
* Firebase / Supabase
* Flutter
* Responsive UI
* Cloud Database

### Future Enhancements
